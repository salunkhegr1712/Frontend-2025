• inline elements
    • inline elements take only what required space and not takes entire width of page
    • inline elements not starts on new line
    • it will not push next block in new line
    • in html its never suggested to put block element inside inline element & but it can contain inline elements
    • you can use css to bend the propertied of inline elements
    • You can use CSS to style inline elements. However, some properties like width and height may not apply.
  • block elements
    • block elements takes entire width of page
    • block elements starts on new line
    • block elements push next content to new line
    • you can use css to bend the propertied of block elements
  • inline-block
    • inline block behavior can be achieved with css
    • inline block is a hybrid structure containing properties of inline + block
    • you can set width height unlike inline element
    • you cant setup width which you can do with block element
    • Can contain other block-level as well as inline elements.
