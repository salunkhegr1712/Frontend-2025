Types of list
  • Unordered list
    • use bullet point to create list & dont have any sequential order
    • you need to use <ul> CONTENT </ul>to declare Unordered list
    • <li> to add new line of list
    • you can use type to change the bullet point symbol

  • Ordered List
    1. it uses numerics or alphabets to list the elements & it has a sequential order
    2. you need to use <ol> CONTENT </ol>to declare Ordered list
    3. <li> to add new line of list
    4. you can use type & start(only works with numbers) to change behavior of list

  • defination list
    • structure
    it uses dictionary like structure to list out the elements
    you need to use <dl> CONTENT </dl>to declare defination list/ container for the list
    <dt> (dt means defination term)to provide heading for list point
    <dd>(dt means defination description) to provide data for heading provided above
